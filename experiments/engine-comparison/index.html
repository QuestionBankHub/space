Pick the engine for "we are in space"

Same scene in each demo — Earth with real NASA day/night textures, a real star catalog (Hipparcos sample), sun lighting, and free orbit camera. Drag to rotate, scroll to zoom. Compare the look and the load time.

Option A — Recommended

Three.js + WebGPU

Custom shaders, biggest ecosystem, most flexible. Highest ceiling for "stunning." My pick.
Option B

Babylon.js

PBR-first, native WebGPU, gorgeous lighting out of the box, smaller astronomy ecosystem.
Option C

CesiumJS

Geospatial truth — real planet surfaces, accurate orbits. Heavier, ellipsoid-centric.
Option D

Hybrid (Three + Cesium)

Three.js for cosmic / orbital views, Cesium when you land on a real planet.
Each demo loads from a CDN — no install. If a demo is slow on first load, that's network, not the engine. Built by Claude in Space/engine-demos/. Delete the folder when you're done choosing.
